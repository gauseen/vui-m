### Masker

> 遮罩层显示

------

#### 例子

基本用法

```html
<Masker v-model="visible">
  <span>自定义内容</span>
</Masker>
```

### API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|------|-------|---------|-------|--------|
| v-model | 赋值一个变量，控制是否显示遮罩 | Boolean | | |

### Slot

#### Masker
| name | 描述 |
|------|--------|
| - | 遮罩层的内容 |
